# ebay-terapeak-mcp documentation

**ebay-terapeak-mcp** is a [Model Context Protocol](https://modelcontextprotocol.io/) server **and CLI** for eBay **Terapeak Product Research** — sold & active listing data — driven by a persistent, logged-in Playwright session. Over **stdio** for Claude Code, Claude Desktop, Cursor, and other MCP hosts, or straight from the terminal.
